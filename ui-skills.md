# UI/UX Redesign Skill

When this skill is active, the agent should follow these specialized workflows for website modernization.

## Core Directives
1. **Competitive Analysis**: When a URL is provided, analyze the layout, typography, and color palette. 
2. **Component Mapping**: Break down the competitor's UI into reusable React/Tailwind components (e.g., Bento Grids, Hero Sections, Stat Cards).
3. **Consistency**: Always incorporate the user's specific brand colors (#98d339, #015ba7, #000c63) while maintaining the competitor's structural layout.

## Implementation Steps
- **Step 1: Audit**: Scan the current codebase for existing UI patterns.
- **Step 2: Plan**: Generate an Implementation Plan artifact that outlines the transition from the old CSS/Component structure to the new "Modern Enterprise" style.
- **Step 3: Execution**: Use Tailwind CSS for rapid styling, ensuring mobile responsiveness and accessibility (WCAG) compliance.
- **Step 4: Verification**: Use the Browser tool to capture screenshots of the new components and compare them against the competitor's visual weight and balance.
